## Overview

Vegan Kök is a responsive website dedicated to sharing delicious and easy vegan recipes. Our mission is to inspire and simplify plant-based cooking for everyone. The site is built using custom HTML5 and CSS3, ensuring an accessible, lightweight, and responsive experience across all devices.

## User Experience (UX)

### Project Goals

#### User Goals
- Discover a variety of easy and delicious vegan recipes.
- Learn the basics of plant-based cooking.
- Access quick, budget-friendly meal ideas.

#### Site Owner Goals
- Promote vegan cooking and healthy eating.
- Provide comprehensive and inspiring recipe resources.
- Build a community of plant-based cooking enthusiasts.

### User Stories
1. **New Vegan:** As a new vegan, I want to find simple recipes that are easy to prepare.
2. **Busy Professional:** As a busy professional, I want to find quick meal ideas.
3. **Budget-Conscious Cook:** As a budget-conscious cook, I want recipes that use affordable ingredients.
4. **Curious Food Lover:** As a curious food lover, I want to explore and learn more about vegan cooking.

### UX Design Rationale
Our design decisions are rooted in a user-centric approach:

- **User-Focused Layout:**  
  The site’s layout is intuitive, ensuring users can quickly locate recipes, understand navigation, and absorb content without distraction.

- **Wireframe-Informed Structure:**  
  Early wireframes guided the placement of essential elements (header, navigation, hero section, content areas, and footer) to create a seamless user journey across desktop, tablet, and mobile devices.

- **Accessibility & Readability:**  
  We emphasize clear typography, high-contrast colors, and logical content structure to enhance accessibility for all users.

- **Iterative Design Process:**  
  Our design evolves based on user feedback and testing, allowing us to continuously refine the experience to meet real user needs.

## Features

### Existing Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop.
- **Interactive Navigation:** An intuitive menu for easy site exploration.
- **Recipe Categories:** Organized into Breakfast, Lunch, Dinner, Snacks, and Desserts.
- **Detailed Recipe Pages:** Each recipe comes with clear instructions and nutritional information.
- **About Us Section:** Information about the vision and mission behind Vegan Kök.
- **Contact Form:** For user inquiries and feedback.

### Future Features
- **Recipe Finder:** “What’s in your kitchen?” feature to suggest recipes based on available ingredients.
- **Nutritional Calculator:** Provides detailed nutritional information for each recipe.
- **User Submissions:** A system for users to submit their own recipes.
- **Meal Planning Tools:** Assistance with planning meals for the week.
